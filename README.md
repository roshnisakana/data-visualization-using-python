# 📊 Customer Purchase Behavior Analysis

## Overview

This project analyzes customer purchasing patterns using Python and data visualization techniques. The objective is to uncover trends in customer spending, product performance, sales growth, and customer segmentation.

Developed as part of the **Data Visualization with Python (24CSE463)** course at New Horizon College of Engineering.

## Objectives

* Analyze customer transaction data.
* Identify top-performing products and categories.
* Study customer spending behavior.
* Perform customer segmentation using RFM Analysis.
* Visualize sales trends and revenue patterns.
* Generate customer insights for business decision-making.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

The project uses a customer transaction dataset containing:

* Customer information
* Product categories
* Purchase details
* Transaction dates
* Payment methods
* Revenue and discount information

## Analysis Performed

### 1. Data Cleaning & Feature Engineering

* Removed duplicate records
* Handled missing values
* Created additional date-based features

### 2. Correlation Analysis

* Analyzed relationships between numerical variables
* Visualized correlations using a heatmap

### 3. Sales Trend Analysis

* Monthly revenue trends
* Monthly order count analysis

### 4. Product & Category Performance

* Top-performing product categories
* Highest revenue-generating products

### 5. Customer Segmentation (RFM Analysis)

Customers were segmented based on:

* Recency
* Frequency
* Monetary Value

Segments include:

* Champions
* Loyal Customers
* Potential Loyalists
* At Risk
* Lost Customers

### 6. Spending Distribution Analysis

* Transaction amount distribution
* Category-wise spending patterns
* Payment method analysis

### 7. City-wise Revenue Analysis

* Revenue by city
* Day-of-week purchasing patterns
* Heatmap visualization

## Project Outputs

* customer_transactions.csv
* customer_summary.csv
* Interactive visualizations and analytical charts

## Key Insights

* Identified revenue-generating product categories.
* Discovered customer purchasing trends over time.
* Segmented customers for targeted marketing strategies.
* Analyzed spending behavior across different customer groups.

1. Clone the repository
2. Install required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open the notebook

```bash
jupyter notebook customer_analysis_1.ipynb
```

4. Run all cells
